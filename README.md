# dotfiles
The configuration files for an Arch Linux rice based around Gruvbox.

+ __GTK Theme:__ [Arc](https://github.com/horst3180/arc-theme) modified with Gruvbox colours  
+ __Launcher:__ rofi
+ __Terminal:__ Alacritty + zsh  
+ __Bar:__ Polybar
