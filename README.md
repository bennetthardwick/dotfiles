# dotfiles
Gruvbox inspired Arch Llinux i3wm rice.
+ __GTK Theme:__ [Arc](https://github.com/horst3180/arc-theme) modified with Gruvbox colours  
+ __Launcher:__ rofi
+ __Terminal:__ RXVT-Unicode + zsh  
+ __Bar:__ Polybar
